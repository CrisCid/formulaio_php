Base de datos mysql utilizando Xampp v3.3.0.
PHP v8.2.0
Buscar donde se instalo Xampp
en mi caso se instalo en C:\xampp\
en esa ruta buscar la carpeta htdocs y crear una carpeta o clonar directamente el proyecto en dicha carpeta
abrir Xampp, iniciar Apache y MySQL, abrir http://localhost/phpmyadmin/ en el navegador para crear la base de datos, en la carpeta "Base de datos" 
esta todo lo necesario para que pueda funcionar el formulario.
para abrir el proyecto en el navegador utilizar la siguiente ruta http://localhost/"carpeta creada"/index.php
Fin.
